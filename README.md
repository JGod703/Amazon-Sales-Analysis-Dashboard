# Amazon-Sales-Analysis-Dashboard
Proyecto de análisis de datos enfocado en productos de Amazon, utilizando Excel, SQL Server y Power BI para explorar patrones entre precios, descuentos, ratings y popularidad de productos.

## Objetivos
- Analizar relación entre precios y ratings
- Identificar categorías más populares
- Explorar impacto de descuentos
- Construir dashboard interactivo

## Herramientas utilizadas
- Microsoft Excel
- Microsoft SQL Server
- Microsoft Power BI

## Data Cleaning
- Corrección de encoding UTF-8
- Eliminación de símbolos monetarios
- Conversión de columnas numéricas
- Creación de main_category
- Revisión de valores nulos

## SQL Analysis
Se realizaron consultas SQL en SQL Server para explorar y analizar el dataset. Entre los análisis realizados se incluyen:

- Exploración inicial de los datos
- Identificación de categorías existentes
- Top 10 productos con mayor número de opiniones
- Productos mejor valorados y con alta popularidad
- Promedio de rating por categoría
- Relación entre descuentos y ratings
- Identificación de productos premium mediante análisis de precios

Las consultas permitieron descubrir patrones relacionados con popularidad, satisfacción de clientes y distribución de productos dentro del catálogo.

## Dashboard
El dashboard fue desarrollado en Power BI e incluye:

- KPIs generales
- Filtros interactivos
- Relación entre precio y rating
- Top productos con mayor número de opiniones
- Distribución de productos por categoría

## Insights encontrados
- Electronics concentra la mayor cantidad de productos
- La mayoría de productos tienen ratings superiores a 4
- No existe correlación fuerte entre precio y rating
- Algunos productos económicos muestran ratings competitivos frente a productos de mayor costo
